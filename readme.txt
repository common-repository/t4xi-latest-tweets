== Description ==

A simple plugin that shows the latest tweets from a list, timeline, a search or user.

== Installation ==

1. Upload the 't4xi-latest-tweets' folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Drag the widget to your sidebar.
4. Fill in the required fields.
5. Save the changes.
6. Enjoy it.
